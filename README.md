# Projeto Barbearia Acadêmico

Uma aplicação web simples para seleção de serviços de barbearia, com cálculo automático do total.

## Funcionalidades

- Seleção de serviços (Corte, Barba, Sobrancelha)
- Cálculo automático do total
- Interface responsiva com animações
- Botão para limpar o total
- Ícones visuais para melhor experiência

## Como executar

1. Abra o arquivo `index.html` em um navegador web.
2. Clique nos botões "Adicionar" para incluir serviços no total.
3. Use o botão "Limpar" para resetar o total.

## Tecnologias utilizadas

- HTML5
- CSS3 (com Flexbox e animações)
- JavaScript (ES6)
- Font Awesome (para ícones)

## Estrutura do projeto

- `index.html`: Estrutura da página
- `main.css`: Estilos e layout
- `main.js`: Lógica de cálculo e interações
